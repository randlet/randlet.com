QATrack+

A tool for managing the QA program of a radiotherapy department
Visit the QATrack+ homepage at qatrackplus.com.
{% endblock %} {% block content %}

QATrack+ is a user friendly, web based system (developed with Python & Django) for entering, reviewing, and trending the data generated by a radiotherapy clinics quality control program.

This is an open source project we are working on at The Ottawa Hospital Cancer Centre and we are currently using QATrack+ to manage the vast majority of our daily, weekly and monthly QA data.
